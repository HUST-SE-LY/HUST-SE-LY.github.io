Dockerfile 指令

RUN

在docker build时运行

copy

如果源路径为文件夹，复制的时候不是直接复制该文件夹，而是将文件夹中的内容复制到目标路径。

ADD

ADD可以使用URL作为路径，对压缩包文件会自动解压缩

CMD

CMD命令指定了容器启动时执行的命令

ENTRYPOINT

同样是在容器启动时执行的命令

ENV

设置环境变量，可以在后续指令中引用

ARG

与ENV作用一致但设置的参数只能在docker build中使用

VOLUME

定义匿名数据卷

EXPOSE

声明端口

WORKDIR

指定工作目录（必须提前创建好）

USER

切换后续命令执行的用户

HEALTHCHECK

指定运行状态

ONBUILD

延迟构建指令（第一次构建不会执行的指令，但之后再以此构建新的镜像会执行）

LABEL

LABEL 指令用来给镜像添加一些元数据（metadata），以键值对的形式
