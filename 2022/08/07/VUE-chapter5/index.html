使用vite创建vue项目

yarn create vite创建vite

随后设置项目名称，选择vue项目，选择vue

创建好项目后，在项目根目录下运行npm install安装依赖

安装好依赖后，运行npm run dev运行项目

首先在子组件中定义emit

然后我们需要在子组件中触发事件

通过绑定click事件触发emit函数，emit函数接收一个或多个参数，第一个参数就是在emit数组中的事件名，后面的可选参数是要传递的值
